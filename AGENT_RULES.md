# AGENT_RULES (Licensing & Attribution)

Это рабочие правила для IDE‑агента.

## Инварианты
- Не менять тип лицензии без прямой команды человека.
- Разделять код и контент по папкам и лицензиям.
- Всегда фиксировать внешние источники и авторство.

## Классификация новых модулей
При появлении новой папки:
1) спросить: ядро / интеграция / контент;
2) добавить пометку лицензии в README;
3) при необходимости обновить `THIRD_PARTY_NOTICES.md`.

## Контроль смешивания слоёв
- Контент в `core/` → предложить вынести.
- Код в `content/` → предложить перенести.

## Авторство и внешние заимствования
- Копипаст: запись в `THIRD_PARTY_NOTICES.md`.
- Форк: “adapted from …”, оригинальная лицензия.
- Идеи: acknowledgements в AUTHORS/README.

## PR‑поддержка
- Классифицировать изменения по лицензии.
- Предлагать добавить авторов при существенном вкладе.
