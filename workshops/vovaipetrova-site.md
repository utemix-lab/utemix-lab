# Мастерская: vovaipetrova-site

## Что делает

Статический сайт-витрина проекта, отображающий контент из базы знаний.

## Зачем нужна в экосистеме

- Витрина проекта для публичного представления
- Разделение мастерской (core) и продукта (site)
- Независимая структура от ядра

## Как работает

- Импорт данных из `vovaipetrova-core`
- Статическая генерация HTML
- Hash-based routing
- GitHub Pages для хостинга

## Связи с другими мастерскими

- **← vovaipetrova-core**: Импорт данных (JSONL, JSON)
- **→ extended-mind**: Возможная интеграция с RAG (планируется)

## Особенности

- Независимая структура в `site-structure.json`
- Компоненты без стилей (только логика)
- Статическая генерация
