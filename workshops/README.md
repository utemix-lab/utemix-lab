# Описания мастерских

Здесь хранятся описания каждой мастерской (репозитория) в экосистеме.

## Структура описания

Каждая мастерская описывается через:
- Что делает
- Зачем нужна в экосистеме
- Как работает
- Связи с другими мастерскими
- Особенности работы
- История разработки

## Файлы

- `vovaipetrova-core.md` — мастерская базы знаний
- `vovaipetrova-site.md` — мастерская статического сайта
- `dream-graph.md` — мастерская визуализации графов
- `extended-mind.md` — мастерская метамодели
- `evoquant.md` — мастерская Evoquant
- `minds-lorgnette.md` — мастерская Minds Lorgnette

## Обновление

Описания обновляются:
- При значительных изменениях в мастерской
- При изменении связей с другими мастерскими
- По запросу из экосистемного слоя

## Лицензирование

Тексты мастерских относятся к контентному слою.
См. `CONTENT_LICENSE.md` в корне репозитория и общие правила в `README.md`.
