# Быстрый старт: экосистемный слой

## Что это

Межрепозитарный слой координации мастерских (репозиториев) без конфликта с внутрирепозитарной работой.

## Структура

```
ecosystem/
├── README.md              # Общее описание
├── ANSWERS.md             # Ответы на вопросы перед началом работы
├── QUICK_START.md         # Этот файл
├── IMPLEMENTATION_STATUS.md # Статус реализации
├── agent-roles.md         # Роли агентов
├── links.md               # Связи между репо
├── data-contracts.md      # Контракты данных
├── .cursorrules           # Правила для агента экосистемы
├── workshops/             # Описания мастерских
├── history/               # История разработки
└── definition-archive/   # Архивы дефиниций
```

## Как использовать

### Для агента мастерской

1. Работай в рамках своего репозитория
2. Если видишь изменения извне — проверь `ecosystem/links.md`
3. Не «чисть» архивы вариантов

### Для агента экосистемы

1. Описывай связи в `links.md`
2. Предлагай изменения через issue/PR
3. Не командуй агентами мастерских

## Ключевые принципы

- **Не диктующий, а координирующий**
- **Гибкий и недооформленный**
- **Архивный подход**
- **Уважение к мастерским**

## Документация
